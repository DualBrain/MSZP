'-----------------------------------------------------------------------
'                             How to use it
'-----------------------------------------------------------------------

It is important to leave everything where it is before the 'How to use'
section. Do not add or remove lines! (except for the palette content
of course)

_____________________________ Color settings ___________________________

Start from:
Color gradient for first iterations. Can be 'black' or 'white'.

Fade to black: 1 = yes, 0 = no

________________________________ Palette _______________________________

Cycle length:
Length (in iterations) of the color cycle. (to remove later)

Number of colors: Must be < or = to number of colors listed below.
Limit of 15 colors allowed. (at least for now)

Colors: [position on palette (from 0 to 1), R value, G value, B value]
It is highly recommended to leave the first color at position 0 as it will
be duplicated at position 1. (for smooth transitions between cycles)

You can use this online tool to visualize the palette before entering the
values in this file. (just remember to duplicate the first color to the
end of the palette)
Tool: https://cssgradient.io/
